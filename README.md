# StudyGenie 📚

AI-powered study assistant that transforms raw notes into summaries, questions, and simple explanations — powered by OpenAI.

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
export OPENAI_API_KEY='your-api-key-here'
python main.py
```

### Frontend
Open `frontend/index.html` in your browser or serve it locally:
```bash
cd frontend
python -m http.server 8000
```

Visit http://localhost:8000
